Code Injection (eval)

Code injection occurs when user input is passed to functions like eval(), exec(), or other code execution functions.

Vulnerable Test Cases

  • JavaScript eval() - Server-side JS eval injection
    require('child_process').execSync('id')
  • PHP eval() - PHP eval injection
    system('id')
  • Python eval() - Python eval/exec injection
    __import__('os').system('id')
  • Expression Evaluator - Math expression evaluation
    Vulnerable to expression injection

False Positives (Safe)

  • Validated Input - Only allows numbers and math operators
