LDAP Injection

LDAP injection occurs when user input is embedded into LDAP queries without proper sanitization.

Vulnerable Test Cases

  • Basic LDAP - User lookup
    * or admin)(|(uid=*)
  • Authentication Bypass - Login via LDAP
    username=* password=*
  • User Search - Search functionality
    *)(objectClass=*
  • Blind LDAP - Boolean-based extraction
    admin* to enumerate

False Positives (Safe)

  • Escaped Input - Special characters rejected
