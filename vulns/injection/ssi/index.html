Server-Side Includes (SSI) Injection

SSI injection occurs when user input is processed by the SSI engine, allowing execution of SSI directives.

Vulnerable Test Cases

  • Basic SSI - Echo directive
    <!--#echo var="DATE_LOCAL" -->
  • Command Exec - Exec directive
    <!--#exec cmd="id" -->
  • File Include - Include directive
    <!--#include virtual="/etc/passwd" -->

False Positives (Safe)

  • SSI Disabled - Directives are escaped
