HTTP Parameter Pollution (HPP)

HPP occurs when duplicate parameters are handled differently by different components (WAF vs app, validation vs execution).

Vulnerable Test Cases

  • Basic HPP - Shows how duplicates are handled
    ?id=1&id=2&id=3
  • Form HPP - Hidden field override via duplicates
    Hidden fields can be overridden
  • WAF Bypass - WAF checks first, app uses last
    ?q=safe&q=malicious

False Positives (Safe)

  • Properly Handled - Rejects duplicate parameters
