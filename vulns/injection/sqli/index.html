SQL Injection Test Cases

Each database has its own set of test cases with database-specific error messages and syntax.

By Database

  • MySQL - SLEEP(), MySQL error syntax
  • PostgreSQL - pg_sleep(), PG error syntax
  • MSSQL - WAITFOR DELAY, SQL Server errors
  • SQLite - SQLite error syntax

Test Case Types

  • Error-Based - Error messages leak database information
  • Union-Based - UNION SELECT extracts data from other tables
  • Blind Boolean - Application behavior differs based on true/false conditions
  • Blind Time - Time delays indicate successful injection
  • FP Parameterized - Safe implementation using prepared statements
