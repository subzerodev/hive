XPath Injection

XPath injection occurs when user input is embedded into XPath queries without proper sanitization, allowing attackers to manipulate XML data queries.

Vulnerable Test Cases

  • Basic XPath - User search via XPath
    ' or '1'='1
  • Authentication Bypass - Login via XPath
    ' or '1'='1' or '1'='1
  • Blind XPath - Boolean-based extraction
    ' and substring(password,1,1)='a

False Positives (Safe)

  • Validated Input - Only alphanumeric allowed
