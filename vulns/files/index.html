Sensitive Files

Exposed backup files, configuration files, and development files.

Backup Files

  • SQL Backup - Database dump exposed
  • Config Backup - config.php.bak
  • Archive Files - .zip, .tar.gz backups
  • Old Files - .old file extensions

Common Sensitive Files

  • .git/config - Git configuration
  • .git/HEAD - Git HEAD reference
  • .env - Environment variables
  • .htaccess - Apache config
  • .htpasswd - Password hashes
  • wp-config.php - WordPress config
  • composer.json - PHP dependencies
  • package.json - Node dependencies
  • docker-compose.yml - Docker config

Development Files

  • phpinfo() - PHP information
  • Debug Page - Debug information
  • Test Page - Test credentials

False Positives (Safe)

  • Protected File - Returns 403 Forbidden
