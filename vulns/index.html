HIVE

Hazardous Intentionally Vulnerable Environment

A vulnerability testbed for validating web vulnerability scanners.

Vulnerability Categories

  • Injection - SQLi, Command, LDAP, XPath, SSI, SSTI, Code, XML
  • Cross-Site Scripting - Reflected, DOM, Stored, Framework
  • File/Path - Traversal, Upload, Source Disclosure
  • Auth/Session - CSRF, Cookies, Password Exposure
  • Info Disclosure - Errors, PII, Keys, Versions
  • Configuration - CORS, CSP, Clickjacking, Headers
  • Serialization
  • SSRF - HTTP, DNS
  • Backdoors

Authentication Testing

  • Auth Flows - Form, AJAX, Multi-step, OAuth, HTTP Basic, JWT

Static Files

  • Directory Listing & File Tests

API

  • GET /health - Health check
  • POST /api/reset - Reset all databases
