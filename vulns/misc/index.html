Miscellaneous Vulnerabilities

Various security issues that don't fit into other categories.

Password Autocomplete

  • Autocomplete Enabled - Browser can store credentials
  • Autocomplete Disabled - SAFE

Cookie Issues

  • Duplicate Cookies - Same name, different paths
  • Single Cookie - SAFE

Resource Loading

  • Path-Relative CSS - Relative stylesheet path
  • Absolute CSS Path - SAFE
  • Cross-Domain Script - External JS includes
  • Vulnerable JS Libraries - Outdated libraries

Header-Dependent Responses

  • Referer-Dependent - Content changes by Referer
  • User-Agent Dependent - Content changes by UA

Caching Issues

  • Cacheable Sensitive - Sensitive data can be cached
  • Non-Cacheable - SAFE

Input Reflection

  • Input Reflected - User input echoed in response
  • Input in Header - Input reflected in response header
