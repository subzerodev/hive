HTTP Methods

Dangerous HTTP methods that should be disabled in production.

Dangerous Methods Enabled

  • PUT Method - File upload capability
    curl -X PUT -d "content" URL
  • TRACE Method - Cross-Site Tracing (XST)
    curl -X TRACE URL
  • DELETE Method - Resource deletion
    curl -X DELETE URL
  • OPTIONS - Permissive - Overly permissive CORS
    curl -X OPTIONS URL
  • Arbitrary Methods - Accepts any method
    curl -X FOOBAR URL

False Positives (Safe)

  • Restricted Methods - Only GET/POST allowed
