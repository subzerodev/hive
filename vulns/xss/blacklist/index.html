XSS Blacklist Filter Bypasses

Test cases for bypassing common XSS blacklist filters. Each level adds more restrictions.

Vulnerable (Bypassable Filters)

Level Filter Bypass
Level 1 Blocks <script> (exact, case-sensitive) <SCRIPT> or <img onerror=>
Level 2 Blocks <script> (case-insensitive) <img>, <svg>, <body> tags
Level 3 Blocks <script>, <img> <svg>, <body>, <input> tags
Level 4 Blocks <script>, <img>, <svg>, <body> <iframe>, <input>, <details> tags
Level 5 Blocks common tags + javascript: HTML entity encoding, data: URLs
Level 6 Blocks common event handlers onpointerover, onanimationend, etc.
Level 7 Blocks "alert" function confirm(), prompt(), eval(), constructor
