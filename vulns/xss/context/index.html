XSS Context-Specific Test Cases

XSS vulnerabilities in different HTML/JavaScript contexts. Each context requires different payloads.

Vulnerable Test Cases

HTML/XML Contexts

  • SVG Context - XSS inside SVG elements
  • Data Attribute - XSS in data-* attributes
  • iframe srcdoc - XSS in iframe srcdoc attribute
  • HTML Comment - XSS breaking out of HTML comments

URL Contexts

  • href Attribute - javascript: URLs in links
  • src Attribute - XSS in image src attributes

CSS Contexts

  • CSS Style - XSS in inline/embedded CSS
  • CSS Expression - expression() in CSS (legacy IE)

JavaScript Contexts

  • JS Comment - XSS breaking out of JS comments
  • JS Template Literal - XSS in template literals

False Positives (Safe)

  • Properly Sanitized - All contexts properly escaped
