Form Hijacking / Link Manipulation

Vulnerabilities where attackers can control form actions or link destinations to steal credentials or redirect users.

Form Action Hijacking

  • Form Action - User-controlled form action
    ?action=https://evil.com/phish
  • Hidden Field - Manipulable hidden return URL
    ?return=https://evil.com/
  • Formaction Attribute - Button formaction override
    ?submit=https://evil.com/

Link Manipulation

  • Link Href - User-controlled link destination
    ?href=javascript:alert(1)
  • Base Tag - Controls all relative URLs
    ?base=https://evil.com/

False Positives (Safe)

  • Validated - Only relative paths allowed
